Online Appointment Management Portal

A simple web-based mini project to manage doctor appointments using HTML, CSS, and JavaScript.
Users can view doctors, book appointments, and see all scheduled bookings. Data is stored using the browser's localStorage.

*Features
View list of doctors (name, specialization, consultation fee)
Book an appointment with:
Patient name
Doctor selection
Date picker
Time (AM/PM format)

*View all booked appointments in a table
Dashboard showing:
Total doctors
Total appointments
Today’s appointments



*How to Run the Project

*Option 1 – Direct Run
Download the project.
Open index.html in any browser.

*Option 2 – VS Code (Recommended)
Open the project folder in VS Code.
Install Live Server extension.

Right-click index.html → Open with Live Server.

(BEST OPTION) 
Option 3 – Python Local Server
just run this command in VS code terminal 

python -m http.server 5500 

Then open:
http://localhost:5500/index.html
